World Wide Web

The world wide web is made up of documents in the html format. They can also have links to images and video as we would see later .

Tags

Tags are used to tell your browser what to display. Tags usually have opening and closing ones but there are exceptions. The text between the tags is called content There are two types of tags:

  1. There are the normal ones that have opening and closing tags and then the content in between them. an example is <p > combined with </p >
  2. Those that have only one tag e.g img tags these are called void tags. These void tags have no content. an example is <img scr="">

About tags

  1. HTML tags create elements, so an opening tag and a closing one together with the content is called an element.
  2. Each tag has a name and a type, that determines what would be created.

Attributes

Note that tags have attributes that e.g link other websites or documents like this Facebook link -> Facebook which is written as <a href="http://facebook.com">Facebook</a>

HTML-CSS-DOM

  • It is important to note that HTML is a language in its own right with its syntax and rules.
  • CSS is another language that is used to change how HTML documents look like in the browser with its own unique syntax and rules.
  • While the DOM is a standard convention for writing html documents.

So note that while HTML determines the structure of the page, CSS determines the style or look of the page while DOM just says which part of the html code should be writen first.
